ID e Class

Class

Existem duas caracteristica do Class:

  • É possivel utilizar o mesmo class em varios e elementos diferentes
  • É possivel utilizar varias classe ao mesmo elemento

ID

Existem duas caracteristicas principais do ID que são contrastantes com as caracteristicas do class:

  • Não e possivel utilizar o mesmo Id em Varios elementos Diferentes
  • Não e possivel ultilizar VÁrios IDs no mesmo elemento

O ID funciona como um garnde CPF. Não podem existir dois iguais, e a mesma pessoa não pode ter varios IDs
